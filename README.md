# Color generator

## Description

A tint and shades color generator app.
Simple and nice user interface design. An application with functionalities that could be useful in other projects.

## Getting Started

### Dependencies

* ReactJS
* SASS

### Executing program

* Render the app in your browser
```
npm start
```
